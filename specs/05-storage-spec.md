# Storage Specification

## Canonical Format
- Markdown
- UTF-8
- Human-readable without specialized tools

## Metadata Philosophy
Metadata should explain *circumstance*, not just classification.

Common fields:
- source
- captured_at
- surface
- capture_mode
- mood (if present)
- tags

## Immutability Rule
The original capture text is immutable.
Interpretations may evolve over time.
