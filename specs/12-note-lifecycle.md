# Note Lifecycle

1. Capture
2. Context Encoding
3. Initial Processing
4. Linking to Existing Hubs
5. Dormancy
6. Re-emergence
7. Hub Formation (if recurring)
8. Re-interpretation
9. Archival

## Hub Formation Rule
A hub is created only when multiple notes naturally converge
around a shared conceptual gravity.

Hubs are discovered, not planned.
