# File Structure

/inbox/
  raw chronological captures

/reflections/
  diary-like entries
  emotional or situational notes

/notes/
  time-bound notes and interpretations

/hubs/
  conceptual gathering places
  long-lived, non-authoritative

/sources/
  articles, books, wikipedia, documents

/projects/
  active lines of inquiry or creation

/archive/
  frozen snapshots of thinking
