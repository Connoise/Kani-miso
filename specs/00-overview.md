# Personal Knowledge, Thought, and Memory Archive

> **Note:** This is the technical specification. For a general introduction, see [`README.md`](../README.md) in the repository root.

## What This Project Is
This project defines a long-term personal system for capturing, processing,
and organizing thoughts, emotions, and external information.

It is designed to function as:
- a temporal archive of attention
- a rhizomatic knowledge graph
- a reflective diary of shifting interests

Meaning and interpretability take priority over efficiency or categorization.

## Core Characteristics
- Rhizomatic (networked, non-hierarchical)
- Temporal (records when and why something mattered)
- Reflective (preserves emotion and uncertainty)
- Durable (Markdown-first, tool-agnostic)
- AI-assisted, not AI-directed

## Notes vs Hubs
- **Notes** are events: snapshots of attention at a moment in time.
- **Hub notes** are places: conceptual gathering points that persist across time.

The system grows through links between notes and hubs rather than folder hierarchy.

## Capture Surfaces
Thoughts may be captured from multiple surfaces:
- Mobile (Telegram)
- Desktop (work or home computer)
- Future interfaces

All capture surfaces feed the same conceptual pipeline.
Differences are preserved as context, not structure.
