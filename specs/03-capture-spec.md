# Capture Specification

## Capture Philosophy
Capture is allowed to be:
- fragmentary
- emotional
- speculative
- unpolished

Clarity is a later concern.

## Capture Surfaces
Each capture may specify a surface:
- mobile
- desktop-work
- desktop-home
- unknown

Surface indicates contextual constraints, not quality.

## Input Types
- Thought
- Reflection
- Question
- Link
- Document
- Image
- Voice

## Recommended Prefixes
- Thought:
- Reflection:
- Question:
- Link:
- Doc:
- Image:
- Voice:

## Optional Inline Context
- Surface:
- Mood:
- Energy:
- Context:
- Trigger:

Surface: mobile  
Mood: nostalgic
