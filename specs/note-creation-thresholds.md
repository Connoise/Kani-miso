# Note Creation Thresholds

## Purpose

Define when a new note should be created versus
when content should be appended or linked.

---

## Create a New Note When

- The content represents a new moment in time
- Emotional or contextual state differs meaningfully
- A new perspective contradicts or reframes an older one
- The content cannot fit without overwriting history

---

## Do NOT Create a New Note When

- Adding clarification to an existing note
- Extending a thought from the same capture session
- Making minor elaborations
- Adding references or links

---

## Preferred Alternative

When uncertain:
- append a section labeled “Later Reflection”
- link to an existing hub
- add a follow-up note referencing the original

---

## Default Stance

Err toward fewer notes,
but never at the cost of historical accuracy.
